# 🌿 GRECA Café Bar — Experiencia Gastronómica Digital

![GRECA OG Image](https://greca-cafebar.com/og-image.png)

**GRECA Café Bar** es una plataforma digital diseñada para elevar el ritual del café y la buena mesa. Enfocada en la elegancia, la fluidez y el sabor, esta aplicación ofrece un menú interactivo premium con una interfaz de usuario envolvente y moderna.

---

## ✨ Características Principales

- ☕ **Menú Digital Interactivo**: Categorías dinámicas (Desayunos, Tardear, Sándwiches, Cafés y Bebidas) con filtrado instantáneo.
- 🎥 **Hero Media Envolvente**: Una experiencia visual de entrada con video de alta calidad y transiciones suaves.
- 📜 **Scroll-Expansion Effects**: Navegación fluida que expande y revela el contenido de forma elegante mientras el usuario explora.
- 💎 **Estética Premium**: Paleta de colores curada (Verde Bosque, Terracota, Crema) y tipografías sofisticadas para una identidad de marca coherente.
- 📱 **Mobile First**: Diseño totalmente responsivo optimizado para una consulta rápida desde cualquier smartphone en el local.
- 🔍 **SEO de Alto Impacto**: Metadatos enriquecidos y estructurados (JSON-LD) para visibilidad en buscadores y previsualizaciones perfectas en redes sociales.

---

## 🛠️ Stack Tecnológico

- **Core**: [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Estilos**: [Tailwind CSS](https://tailwindcss.com/)
- **Animaciones**: [Framer Motion](https://www.framer.com/motion/)
- **Iconografía**: [Lucide React](https://lucide.dev/)

## 🚀 Inicio Rápido

### Requisitos Previos
- Node.js (v18+)
- pnpm (recomendado) o npm

### Instalación
```bash
# Instalar dependencias
pnpm install

# Iniciar servidor de desarrollo
pnpm run dev
```

### Construcción
```bash
# Generar build de producción
pnpm run build

# Previsualizar build
pnpm run preview
```

## 📍 Local SEO & Datos del Negocio
El proyecto incluye un esquema `JSON-LD` en el `index.html` para mejorar el posicionamiento local. Si necesitas cambiar la ubicación o el teléfono:
1. Abre `index.html`.
2. Busca la sección `<script type="application/ld+json">`.
3. Actualiza `streetAddress`, `telephone` y `geo`.
